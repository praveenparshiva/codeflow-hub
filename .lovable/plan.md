

# CodeFlow – Developer Blog Website

A modern, dark-themed, developer-focused blog/showcase website for your final year project presentation.

## Design System
- **Theme**: Dark background with blue-purple/cyan-indigo gradient accents
- **Style**: Glassmorphism cards, rounded corners, subtle hover effects
- **Typography**: Inter for body text, JetBrains Mono for code elements
- **Animations**: Smooth scroll-triggered fade-in/scale-in animations throughout

## Page Sections (Single-Page Layout)

### 1. Hero Section
- Large title "CodeFlow – Platform for Developers" with gradient text effect
- Tagline and short description with entrance animations
- Custom logo using a `</>` code bracket merged with chat bubble icon (gradient styled)
- Team member names displayed in a clean row/grid
- Call-to-action button scrolling to project overview

### 2. Abstract Section
- Academic-style glassmorphism card with the full abstract text
- Clean typography with proper paragraph formatting

### 3. Project Overview (Infographic)
- Horizontal/grid layout with 5 feature blocks: Authentication, Code Editor, Chat System, Type Speed Test, AI ChatBot
- Each block with an animated icon, title, and bullet points
- Connected with a subtle flow-line design

### 4. Existing vs Proposed System
- Two-column split comparison layout
- Left card (red/muted): Existing system drawbacks
- Right card (green/accent): Proposed system advantages
- Checkmark and cross icons for visual clarity

### 5. Technology Stack
- Icon grid layout organized by category (Frontend, Backend, Integrations)
- Tech logos/icons with labels and subtle hover scale effects

### 6. Team Roles & Contributions
- Four professional profile cards with glassmorphism styling
- Each card shows name, role focus, and contribution bullet points
- Avatar placeholders with gradient borders

### 7. Code Editor Deep Dive
- Dark-themed mockup section resembling a code editor
- Feature list: syntax highlighting, multi-language support, auto-save, terminal output, theme customization
- Terminal-style visual presentation

### 8. Real-Time Chat System
- Feature explanation cards for token generation, channel creation, threaded messaging, video calls
- Animated UI preview placeholders with glassmorphism frames

### 9. Project Documentation
- Clean section with a styled button placeholder for PDF link
- Space reserved for Google Drive PDF integration

### 10. Demo Section
- Two bordered media cards: one for video demo, one for audio explanation
- Placeholder embed areas ready for YouTube/Drive links

### 11. Conclusion
- Closing statement in a highlighted gradient card

### 12. Footer
- Copyright notice: © 2026 CodeFlow Project
- Developer names
- Social media icon placeholders (GitHub, LinkedIn, Twitter)

## Technical Approach
- Single-page React application with smooth scroll navigation
- Fixed top navbar with section links
- All content is static (no backend needed)
- Intersection Observer for scroll-triggered animations
- Fully responsive for mobile, tablet, and desktop
